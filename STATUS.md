# Project Status

**Maintained by:** Erica Walters (solo founder)
**Maintenance level:** Best-effort
**Response time:** No SLA — critical bugs prioritized

This is the open-source MCP client for Dilix, a regulatory intelligence
platform for real estate. It is maintained as a side surface to a paid API
product, by a solo founder.

## What you can expect

- ✅ Critical bugs (broken builds, security issues) get fixed
- ✅ Pull requests reviewed within 1-2 weeks
- ✅ Issues acknowledged within 1-2 weeks
- ⚠️ Feature requests are not promised — we may say no
- ⚠️ Deep refactors / architecture changes are unlikely to be accepted

## What we can't promise

- Same-day response to issues
- Custom feature work for free
- Backwards compatibility forever (we'll deprecate gracefully)

## How to help

- ⭐ Star the repo — it helps us understand demand
- 🐛 File well-scoped bug reports with reproductions
- 💬 Share what you're building with Dilix in Discussions
- 🛠️ Submit small, focused PRs (large PRs get stuck)
